Название проекта:
"Сайт доставки роллов"

Цель проекта: Разработка сайта доставки роллов, используя технологии: flask, sqlalchemy, bootstrap для верстки

Требования к функционалу проекта:
1) Сайт работает, размещен на сервере
2) Есть регистрация и авторизация
3) Загрузка и использование файлов
4) Имеется API
5) Хранение данных в БД
6) Есть шаблоны
7) Есть requirements.txt

Требования к используемым технологиям:
1) flask
2) bootstrap
3) sqlalchemy
4) sqlite3
5) rest API

Сайт должен иметь возможность заказа товаров, возможность положить товары в корзину, управление заказами и отправку деталей
заказа на email.
